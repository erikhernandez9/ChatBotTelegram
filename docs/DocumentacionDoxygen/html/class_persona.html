Detailed Description

Representa a todos los usuarios que utilizan la aplicacion (incluyendo al administrador). Es la clase padre de Usuario, por lo que se usó HERENCIA. El principio OCP es utilizado ya que esta clase se implementa mediante herencia en Usuario. Además se utilizaron los patrones y principios de DIP e ISP, ya que esta clase implementa la interfaz de IUser, esto para darle robustez al código.
