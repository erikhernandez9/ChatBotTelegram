Detailed Description

Clase que almacena todas los empleadores del programa, permitiendo agregarlos, eliminarlos y seleccionarlos. Aplicamos el patrón Creator y el patrón Expert de forma que esta clase es la responsable de crear, eliminar y manipular empleadores. Esto es porque utiliza instancias de la clase Empleador, por lo tanto, podemos decir que la contiene o la agrega, además de ser la clase experta, es decir, tiene la información necesaria para realizar la creación del objeto.
