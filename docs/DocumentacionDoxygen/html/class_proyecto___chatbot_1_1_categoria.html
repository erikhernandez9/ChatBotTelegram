Detailed Description

Esta clase representa al categoria de cada servicio. Se utilizó el principio SRP, ya que la clase contiene solo a la categoría, y tiene la responsabilidad de corroborar si la misma es válida cuando se crea. Además, es la experta en información, ya que solo ella tiene y conoce la información necesaria para crear una categoría.
