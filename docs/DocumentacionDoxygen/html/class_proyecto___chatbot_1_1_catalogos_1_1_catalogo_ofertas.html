Detailed Description

Clase que representa el catalogo de ofertas, en esta se almacenan todas las ofertas, permitiendo agregarlas, eliminarlas y buscarlas segun algunos filtros. Aplicamos el patrón Creator y el principio Expert de forma que esta clase es la responsable de crear, eliminar y manipular ofertas. Esto es porque utiliza instancias de la clase OfertaLaboral, por lo tanto, podemos decir que la contiene o la agrega, además de ser la clase experta, es decir, tiene la información necesaria para realizar la creación del objeto.
