Detailed Description

Clase que almacena todas las contrataciones, permitiendo agregar, eliminar y buscar contrataciones. Aplicamos el patrón Creator y el principio Expert de forma que esta clase es la responsable de crear, eliminar y manipular contrataciones. Esto es porque utiliza instancias de la clase Contratacion, por lo tanto, podemos decir que la contiene o la agrega, además de ser la clase experta, es decir, tiene la información necesaria para realizar la creación del objeto.
