\doxysubsection{Detailed Description}
Un \char`\"{}handler\char`\"{} del patrón Chain of Responsibility que implementa el comando \char`\"{}\+Agregar empleador\char`\"{} y agrega un empleador con todos sus datos. Este handler es un USER STORY. Permite que un usuario se registre en la plataforma, ingresando el mensaje \char`\"{}\+Quiero registrarme como empleador.\char`\"{} Se solicitan todos los datos personales necesarios durante el registro y se guardan en una lista de empleadores. 

\char`\"{}\+Como empleador, quiero registrarme en la plataforma, indicando mis datos personales e información de contacto para  que de esa forma, pueda proveer información de contacto a quienes quieran contratar mis servicios.\char`\"{}
