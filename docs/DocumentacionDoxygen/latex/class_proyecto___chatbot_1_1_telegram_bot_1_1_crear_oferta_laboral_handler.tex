\doxysubsection{Detailed Description}
Handler del patrón Chain of Responsability, procesa el mensaje \char`\"{}\+Crear oferta laboral\char`\"{} y crea una oferta, preguntando por la categoría, el precio y una descripción. 

Este handler pertenece a las USER STORIES. \char`\"{}\+Como trabajador, quiero poder hacer ofertas de servicios; mi oferta indicará en qué categoría quiero publicar, tendrá una descripción del servicio ofertado, y un precio para que de esa forma, mis ofertas sean ofrecidas a quienes quieren contratar servicios.\char`\"{}
